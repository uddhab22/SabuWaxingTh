# Bliss Beauty Studio Website

This is the official website for Bliss Beauty Studio, a professional beauty salon specializing in waxing, sugaring, threading, and facial treatments.

## Features

- Responsive design for all devices
- Online booking system
- Service information and pricing
- Photo gallery
- Client testimonials
- Contact information with map
- FAQ section

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome icons
- Google Maps embed
